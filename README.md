# Theseus
A game launcher which can be used as a CLI, GUI, and a Rust crate for creating and playing Modrinth projects.

Theseus aims to provide three components:
- A **Rust crate** (theseus)
- A **CLI** (theseus-cli)
- A **GUI** (theseus-gui)
